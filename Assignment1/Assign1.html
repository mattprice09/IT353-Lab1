SEO stands for Search Engine Optimization. It is extremely important for businesses because it is the core of web visibility. SEO is essentially the degree to which you make your website visible and relevant to search engines. If you do well with your SEO through the use of solid keywords, credible site-to-site links, busy traffic, and easy crawlability for spiders, then you will have a good shot at getting to the first page on search results. The first, more basic article is an article meant to introduce newbies on the basics of how the web works. The second article is a more detailed article whose purpose is to explain exactly how the web works. One of the things I noticed right away is the presence of GET and POST explanations in the second article. This information is vital to developing web applications as it is the core of how web communication works. I also noticed that they mentioned browser redirects from http://facebook.com to http://www.facebook.com. The article states that this helps with SEO. This is something that I am going to take into account and implement immediately into a couple websites I'm managing. Lastly, I was not aware that the response in the GET request was a small-sized zip file containing the HTML, with the rest of the GET requests for pictures and whatnot coming afterwards. This must be part of the reason why the JavaScript files should be initialized at the end of the HTML document. Cloud Computing is essentially a form of computing in which data and resources are made available to computers on-demand. The biggest difference is that the cloud vastly reduces infrastructure costs for companies because it doesn't require an enormous data center like traditional data centers do. Furthermore, cloud computing allows companies or users to get their projects online faster and keep them going with much less maintainence. Web Application Frameworks are frameworks for software whose main purpose is to reduce overhead of web application. Not only that, but most frameworks simply make software developers' jobs easier by providing prewritten code. This allows developers to use frameworks to that can reduce potentially hundreds, or thousands, of lines of code into less than 100 lines. Some examples include Flask, Meteor, and Django.
